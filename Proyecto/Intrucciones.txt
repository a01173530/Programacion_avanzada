1.- Para compilar este proyecto es necesario: 

gcc shell.c -o shell

2.- Para ejecutarlo

Para modo shell
./shell

o 

para ejecutar lo que hay adentro de un archivo

./shell [batchFile]

Nota: para salir de modo shell se utiliza:
quit
